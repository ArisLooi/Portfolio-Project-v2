# portfolio-project-sample

Welcome to portfolio-project-sample project! This is a sample portfolio website that showcases projects and provides contact information. It's a great starting point for learning HTML, CSS, and JavaScript.

## Features

- Responsive design for different screen sizes
- Showcase of projects with descriptions and images
- Contact form for getting in touch

## Getting Started

### Prerequisites

To run this project, you need a web browser.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ArisLooi/portfolio-project-sample.git
2. Navigate to the project directory:
   ```bash
   cd portfolio-project-sample

## Usage
1. Open the index.html file in your web browser.
2. Browse the home section, view projects, and use the contact form to get in touch.
